# ШуКоманда - Инструкция по запуску

Для запуска фронтенд-части приложения выполните следующие шаги:

1. Перейдите в папку фронтенда:
   ```bash
   cd frontend
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

После этого приложение будет доступно по адресу, указанному в терминале (обычно `http://localhost:5173`).

### Особенности интерфейса
- Приложение автоматически отображается в **мобильном контейнере**. Вам не нужно вручную менять размер окна браузера.
- Все стили разделены по страницам и компонентам для удобства поддержки.
- Папка `backend` создана в корневом каталоге и готова к будущей реализации серверной части.
